Lazy Car (Self Driving Robot)

A self driving robot controlled by a PIC microcontroller and equipped with phototransistors, a thermistor and a hall effect sensor. The robot uses step detection to remain on track and is capable of detecting electro-magnetic fields and sampling temperatures along its course.

RTX Space Attack!

A space battle video game developed on the Keil LPC1768 evaluation board. The game uses the RTX-RTOS to implement and manage concurrent threads for the various components of the game and players control their space craft using the onboard push buttons and joystick.

The Leap Jeep

The "Leap Jeep" was a RC car that was modded to allow users to drive using only hand gestures with the Leap Motion and arduinos to control the circuits. There was also a camera attached to the car that would allow users to get a "life like" driving experience using the Oculus VR headset. This was our entry at Hack Western.

Linear Variable Differential Transformer

An electro-mechanical transducer used to measure displacement through differential change in induced EMF. An Arduino collects and processes the output and displays the final displacement with down to 3% uncertainty

Where are you?

"Where are you" is a web application created to help families and friends find loved ones in the case of a natural disaster, war, terrorist attack or any other disaster. The application allows family or friends to upload pictures of missing individuals to our database where they will be cross referenced using a facial recognition API with images of other "found" people uploaded by helpful civlians. If a close enough match is found, both parties are notified via email and location details can be exchanged. This application thus allows any civilian to easily aid in the search for missing people.

This application was made as an entry at Boston Hacks 2015 and won the sponsor prize from GE for best "Impact Hack".

Bright Box

This project was created at Uoft Hacks 2016 and was awarded first place.

Inspired by the google carboard desgin, the Bright Box is a smart light switch that allows users to control their light switches via our android application. In addition to control, there is also functionality to set alarms to turn on the light in the mornings and ring the built in buzzer as well as to set daily schedules to automate lights. All commands to the switch are sent using HTTP requests which means you can control the lights from anywhere in the world, as long as there is internet access.

The Bright box is purely mechanical and will fit over most standard light switches. This allows easy installation for anyone and does not require any electrical experience.
